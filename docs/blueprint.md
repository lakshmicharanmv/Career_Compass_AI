# **App Name**: Career Compass AI

## Core Features:

- AI Career Advisor: Intelligent career path mapping based on user’s academic stage (10th, 12th, UG) using a custom AI model.
- Career Progression AI: LLM-powered engine that provides career progression mapping, upskilling suggestions, and cross-industry role shifts for professionals.
- AI Resume Reviewer: Accepts uploaded resumes (PDF/DOC), checks for formatting, ATS keyword gaps, grammar issues, and generates corrected, ATS-friendly versions.
- Resume Builder: Creates new resumes from user profile data with industry-specific sections (e.g., Clinical Experience for Doctors, Portfolio for Designers) and diverse templates (Modern, Minimalist, Corporate).
- User Profile: Central hub for users to maintain personal info, education history, skills, achievements, certifications, and career goals.
- Career Dashboard: Visual dashboard showing job trends, salaries, demand for skills, and labor statistics based on selected industry & location.
- AI-Powered Assessment System: Generates dynamic MCQs (20–30) based on user level (10th, 12th, UG, Pro) to provide personalized recommendations.
- AI Chatbot / Career Counselor: A 24/7 AI assistant to answer career questions, give instant resume advice, and guide users through the platform.
- Multi-Level User Flow: Supports 10th students, 12th students, UG students, and working professionals with custom flows (e.g., stream selection, degree recommendations, job role suggestions).
- Industry-Wide Career Support: Works across all fields (Medicine, Law, Arts, Business, IT, Government, Hospitality, etc.), not just tech roles.
- Data Persistence & Navigation: Saves all user data for resume building, test analysis, and future use; provides smooth stepwise navigation (“Next/Back” flow).

## Style Guidelines:

- Primary Color: #1E3A8A – A deep royal blue for trust, stability, and professionalism (darker than #4682B4 for a premium look).
- Background Color: #F9FAFB – A neutral off-white for modern, clean readability.
- Accent Color: #F97316 – A refined orange for CTAs & important highlights (less “neon,” more business-oriented).
- Neutral Shades: Use a gray scale for balance: #111827 (Text – Dark Gray almost black), #4B5563 (Secondary text – muted gray), #D1D5DB (Borders & dividers)
- Headlines: Space Grotesk OR Poppins – A strong, modern sans-serif for titles and key sections.
- Body Text: Inter or Lato – Professional, legible, and widely supported.
- Font Weights: Headlines: 600–700 (bold but clean), Body: 400–500 (medium for readability)
- Use CSS Grid for main layout (clean and modular).
- Use Flexbox for internal components.
- Spacing system: 4px base scale → margins/padding in multiples (e.g., 4px, 8px, 12px).
- Containers: Max width ~1200px with plenty of white space.
- Primary Buttons: Deep Blue background (#1E3A8A), white text, subtle shadow.
- Hover: Slight darkening + animated underline.
- Secondary Buttons: Outlined style (border in deep blue, hover fill).
- CTA Buttons (Important): Accent Orange (#F97316), bold text.
- Use a consistent icon pack (e.g., Lucide Icons or Feather Icons) – thin-line, modern style.
- Icons for career paths, resumes, dashboard widgets, AI assistant, tests – all monochrome (deep blue) with accent color only for alerts/notifications.
- Glassmorphism Cards: Semi-transparent panels with soft blur for key sections like dashboard widgets.
- Micro-interactions: Buttons with smooth hover animations, Cards lift slightly (box-shadow) when hovered.
- Transitions: 200–300ms ease-in-out for smooth feel.
